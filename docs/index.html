A Study of Modern Linux API Usage and Compatibility

This is an API study that reassuringly answers the questions of system developers, from planning stage to your operating system's release. Featuring an API-by-API look at the growth of your operating system and complete chapters on development strategies, evaluation, maintaining APIs — and more.
